# insurance_management_system

Steps to execute the project:
1. Download xampp software and start Apache and MySQL module
2. Download the project and save it in this path C:\xampp\htdocs
3. Go to xampp and click on ADMIN in MySQL
4. phpMyAdmin page will open. Create database "lims" and import the lims.sql file
5. Open new tab in browser and give the url: http://localhost/lims/ 
6. The insurnace management login page will open
7. Admin username: 2001, password: 2211
8. Agent username: 33, password: 444 (can be any given database)
